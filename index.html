Analisis Performa Load Balancing pada Web Server di Microsoft Azure Menggunakan Algoritma Least Connection dan Round Robin

Komponen Utama

Microsoft Azure

Platform cloud dari Microsoft yang menyediakan layanan komputasi, penyimpanan, dan jaringan untuk membangun serta mengelola aplikasi secara global.

Docker

Teknologi container yang memungkinkan aplikasi dijalankan secara konsisten di berbagai lingkungan, memudahkan deployment dan skalabilitas.

Algoritma Least Connection

Metode load balancing yang mengarahkan trafik ke server dengan koneksi aktif paling sedikit, sehingga beban server lebih merata.

Algoritma Round Robin

Metode load balancing yang membagi trafik secara bergiliran ke setiap server secara berurutan, sederhana dan efektif untuk distribusi awal.

Azure Load Balancer

Layanan Microsoft Azure yang secara otomatis membagi trafik ke beberapa server untuk memastikan aplikasi tetap tersedia dan responsif.

Azure Monitor

Fitur pemantauan dari Azure untuk memonitor performa aplikasi dan infrastruktur secara real-time serta mendeteksi masalah lebih awal.
